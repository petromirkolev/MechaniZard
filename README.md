# Mechanizard

Mechanizard is an app for motorcycle owners who do their own maintenance and repairs.

## Current direction

Mechanizard v1 is focused on motorcycles only.

Core areas:

- Garage
- Maintenance logs
- Repair logs
- Guides
- Profile

## Planned v1 features

### Auth

- Register
- Login

### Garage

- Add motorcycle
- Edit motorcycle
- Delete motorcycle with confirmation
- View motorcycle cards with general info
- Quick odometer update

### Maintenance

- Create maintenance logs
- View recent maintenance entries
- View maintenance history

### Repairs

- Create repair logs
- View recent repair entries
- View repair history

### Guides

- Static maintenance guides
- Static repair guides

### Profile

- Update username
- Update email
- Add/edit name
- Add/change avatar

## Tech stack

Planned stack:

- Vite
- Vanilla TypeScript
- Node.js
- Express
- PostgreSQL
- Playwright
- Docker
- GitHub Actions

## Project status

This project is currently in early UI structure and styling phase.

The current focus is:

1. Static HTML structure
2. Shared CSS system
3. Basic screen switching
4. Frontend logic
5. Backend/API
6. Database integration
7. Testing
8. Deployment

## Design direction

Mechanizard is being designed as:

- desktop-first
- dark UI
- glassy / modern panels
- practical and structured, not flashy

## Why this project exists

This project is meant to help me practice building a more realistic app end-to-end while keeping scope under control.

It also serves as a long-running portfolio project covering:

- frontend structure
- UI systems
- TypeScript architecture
- backend/API work
- database design
- E2E testing
- deployment

## Notes

This is an active work in progress.  
The scope will be expanded carefully and only after each foundation layer is stable.
